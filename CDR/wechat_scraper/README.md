# Set up Virtual Environment
## Windows
`pip install -r windows_requirements.txt`
## Linux
`pip install -r linux_requirements.txt`


# Commands to scrape with the spider
## Command line output
`scrapy crawl <spider name>`
## File output
`scrapy crawl <spider_name> -O <file_name>`
